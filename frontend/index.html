ABOUT THE SITE

This page was created using HTML and is hosted in a GitHub repository. The code, including the JavaScript for the visitor counter and the Azure Function, was tested locally. This is how the front end was built. For the back end, we set up an Azure Cosmos DB and container to manage the data. The serverless function interacts with the database to store the counter data. After being developed locally in Visual Studio Code, the Azure Function was deployed to Azure. The static site was also deployed on Azure using a blob container. Subsequently, a Content Delivery Network (CDN) was established for HTTPS support, along with custom domain capabilities. I would love to hear your thoughts!
